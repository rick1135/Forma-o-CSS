Pseudo-elementos

Usado para adicionar elementos extras na pãgina, sem a necessidade de declarà-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::first-letter

Seleciona a primeira letra da primeira linha

Pripriedades disponíveis

  • font
  • color
  • background
  • margin
  • padding
  • border
  • text-decoration
  • vertical-align
  • text-transform
  • line-height

Pseudo-elemento - ::first-line

Seleciona a primeira linha do elemento

Pripriedades disponíveis

  • font
  • color
  • background
  • margin
  • padding
  • border
  • text-decoration
  • vertical-align
  • text-transform
  • line-height

Pseudo-elemento - ::backdrop

Estiliza o plano de fundo de elementos em modo de tela cheia

Pseudo-elemento - ::marker

Estiliza o marcador de lista

Propriedades aceitas:
  • Item 1
  • Item 2
  • Item 3

Pseudo-elemento - ::placeholder

Um placeholder é uma dica para ajudar no preenchimento de algum campo de formulário.

Propriedades aceitas:
  • color
  • font
  • background
  • text

Pseudo-elemento - ::selection

Permite estilizar a seleção de textos no navegador

Propriedades aceitas:
  • color
  • background-color
  • text
